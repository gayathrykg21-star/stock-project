MTI_SMS

Total Stock Management System — PHP Project

About This Project

MTI_SMS is a complete PHP-based stock management system designed for college administration. It provides comprehensive inventory tracking, stock operations, and request management with role-based access control.

Setup Instructions

Step 1: Database Setup

  1. Create a MySQL database named mti_sms
  2. Import public/database/mti_sms.sql via phpMyAdmin or command line

Step 2: Configure Database Connection

  1. Open public/config/database.php
  2. Update DB_HOST, DB_USER, DB_PASS as needed

Step 3: Deploy

  1. Copy the public/ folder contents to your web server
  2. Access via http://localhost/mti_sms/ or your domain
